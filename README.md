# ncmdump-wasm

Netease Music `.ncm` file unlock.

## Run

Requires tool: `cargo-make`, install it using:
> cargo install cargo-make

Then build & run:
> cargo make serve

Open `http://127.0.0.1:4000` to use.

## Online

Live demo: [ncmdump-wasm-online](https://starccy.github.io/ncmdump-wasm)
